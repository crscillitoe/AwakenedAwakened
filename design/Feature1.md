# Summary

You are a developer tasked with implementing a Runescape plugin for the popular Runescape client, Runelite.

You can find the Runelite API documentation here: https://static.runelite.net/runelite-api/apidocs/

We are creating a plugin which will enhance boss fights in the game by adding in additional mechanics.

The first feature we are going to implement is the item that players will use to activate these boss challenges.

In the normal game of Runescape, there is an item called the `Awakener's Orb`. Our plugin should replace this item with a new item called the `Awakener's Cube`.

# Success Criteria

1. The `Awakener's Orb` is now referred to as the `Awakener's Cube` in game.
